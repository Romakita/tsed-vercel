## Installation

```sh
yarn install
```

## Tasks

> **Important!** Ts.ED requires Node >= 12, Express >= 4 and TypeScript >= 4.

```batch
# install dependencies
$ yarn install

# serve
$ yarn start

# build for production
$ yarn build

# deploy on vercel
$ yarn deploy
```
